## 2. 测试点清单

### UT: 数据模型 (18 例)
#### ServerConfig
- [x] 默认值 — port=7890, auto_open_browser=True
- [x] 自定义值

#### LLMConfig
- [x] 默认值 — provider=deepseek, model=deepseek-chat, threshold=6 ...
- [x] api_key 赋值
- [x] provider + base_url 自定义

#### BrowserConfig
- [x] 默认值 — headless=False, slow_mo=500, viewport=1920×1080
- [x] headless=True

#### PlatformConfig
- [x] 默认值 — enabled=False, 空 session_file, 空 base_url

#### TaskConfig
- [x] 必填 `name` 校验 — 不传则 ValidationError
- [x] 默认值 — platforms=[], topic="", schedule="manual", limit=20
- [x] 全字段自定义

#### Config (顶层)
- [x] 空构造 — 所有子模块为默认值
- [x] `model_validate(dict)` — 部分字段
- [x] `model_validate(dict)` — 完整字段含 platforms/tasks
- [x] JSON 序列化 — `model_dump()`
- [x] platforms 字典类型动态赋值
- [x] tasks 列表类型动态追加

### UT: 工厂 (3 例)
- [x] `create(None)` — 返回 YamlConfigLoader（需 monkeypatch DEEPSEEK_API_KEY）
- [x] `create(str)` — 返回 YamlConfigLoader 指定路径
- [x] 返回类型是 `ConfigLoader` 子类

### UT: env 解析 (6 例)
- [x] `${VAR}` 已设置 — 正确替换
- [x] `${VAR}` 未设置 — 抛出 ValueError
- [x] 无占位符字符串 — 原样返回
- [x] dict 递归替换 — 嵌套 dict 中替换
- [x] list 递归替换 — 列表中替换
- [x] 非字符串值 — 原样返回

### TA: YamlConfigLoader 全链路 (12 例)
- [x] 文件不存在 → `Config()` 默认值
- [x] 完整 YAML → 各字段正确解析
- [x] platforms dict 解析
- [x] tasks 列表解析
- [x] `reload()` — 修改文件后重新加载
- [x] `config` property — load 后正常访问
- [x] `${ENV_VAR}` 在顶层字段中替换
- [x] `${ENV_VAR}` 在嵌套 dict 中替换
- [x] `${ENV_VAR}` 在 list 中替换
- [x] `config` property 在 load 后不抛异常
- [x] 空 YAML 文件 → `Config()` 默认
- [x] `null` YAML 文件 → `Config()` 默认

## 3. 公共依赖

- `monkeypatch` — 注入 DEEPSEEK_API_KEY 环境变量
- `tmp_path` — 隔离 YAML 文件系统
- `sample_yaml_content` / `sample_yaml_file` fixture（conftest）

## 4. 运行命令

```bash
pytest tests/wheel/config/ -v
```
