## 2. 测试点清单

### UT: 工厂 (4 例)
- [x] `create(None)` — 返回 JsonlStorage 默认 data_dir="data"
- [x] `create(StorageConfig(data_dir="/tmp"))` — 传入 config 对象
- [x] `create("/custom/path")` — 传入字符串路径
- [x] 返回类型是 `Storage` 接口子类

### TA: JsonlStorage 全链路 (19 例)

#### 生命周期
- [x] `name` — "JsonlStorage"
- [x] `start()` — 创建 data 目录，health_check=True
- [x] `start()` 幂等 — 两次 start 不抛异常
- [x] `stop()` — 不抛异常
- [x] `health_check` — start 前 False，start 后 True

#### items 读写
- [x] `save_items` + `load_items` — 写入 3 条，读出 3 条
- [x] `load_items` 空目录 — 返回 `[]`
- [x] `load_items` 平台筛选 — platform="twitter" 只返回 twitter 条目
- [x] `save_items` 追加 — 两次写入同一日期，读取合并
- [x] `load_items` 筛选不存在的平台 — 返回 `[]`

#### report 读写
- [x] `save_report` + `load_report` — 往返一致
- [x] `load_report` 缺失 — 返回 None
- [x] `save_report` 覆写 — 两次写入，第二次覆盖

#### summary 读写
- [x] `save_summary` + `load_summary` — 往返一致（含嵌套 dict）
- [x] `load_summary` 缺失 — 返回 None

#### list_dates
- [x] 空目录 — 返回 `[]`
- [x] 倒序排列 — 03 > 02 > 01
- [x] sessions 目录过滤 — 不出现 "sessions"

#### 完整性验证
- [x] LLM 字段完整往返 — 含全字段的 item 写入 → 读取 → 逐字段对比
- [x] 多日期隔离 — 日期A 和日期B 数据互不干扰

## 3. 公共依赖

- `StorageFactory` + `StorageConfig` — 创建被测实例
- `UnifiedItem` — 构造测试数据
