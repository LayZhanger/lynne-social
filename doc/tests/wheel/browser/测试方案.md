# wheel/browser 测试方案

## 1. 被测目标

| 源文件 | 类型 | 测试文件 |
|--------|------|---------|
| `browser_models.h/.cpp` — `BrowserConfig` | UT | `test_browser_models_ut.cpp` |
| `imp/cdp_browser_manager.h/.cpp` — `CdpBrowserManager` + `BrowserContext` | TA | `test_browser_ta.cpp` |
| 所有 BrowserContext 接口（含人机交互） | TA | `test_browser_baidu_ta.cpp` |
| 人机交互 7 方法（click/type/scroll/press_key/hover/exists/wait_for_selector） | TA | `test_browser_human_ta.cpp` |
| 人机交互边界/错误路径 | TA | `test_browser_human_ta_edge.cpp` |
| 跨站验证（小红书 xiaohongshu.com） | TA | `test_browser_xiaohongshu_ta.cpp` |
| `browser_helpers.h/.cpp` — `dump_page_structure()` | 工具 | 由 `test_browser_xiaohongshu_ta` 隐式测试 |

## 2. 测试点清单

### UT: BrowserConfig（6 tests）

| # | 名称 | 验证内容 |
|---|------|---------|
| 1 | DefaultValues | 所有字段有合理默认值：headless=true, slow_mo_ms=500, timeout_ms=30000... |
| 2 | CustomValues | 全字段自定义后读回正确 |
| 3 | ToJson | to_json 输出包含所有字段 |
| 4 | FromJson | from_json 正确解析有效 JSON |
| 5 | EmptyJson | 空 JSON → 全用默认值 |
| 6 | PartialJson | 部分字段 → 缺的字段用默认值 |

## 3. 运行命令

```bash
# UT
dist/bin/test_browser_models_ut

# TA (全量)
dist/bin/test_browser_ta
dist/bin/test_browser_baidu_ta
dist/bin/test_browser_human_ta
dist/bin/test_browser_human_ta_edge
dist/bin/test_browser_xiaohongshu_ta

# 全量回归（含其他模块）
ctest --test-dir build --output-on-failure -j$(nproc)
```
