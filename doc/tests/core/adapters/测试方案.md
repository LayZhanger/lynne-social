# Adapter 测试方案

> 注意：本模块尚未实现，以下为计划中的测试方案。

## 测试范围

| 文件 | 类型 | 说明 |
|------|------|------|
| `test_adapter_models_ut.cpp` | UT | AdapterConfig 默认值/自定义 |
| `test_adapter_factory_ut.cpp` | UT | AdapterFactory create/未知平台 |
| `test_rednote_adapter_ta.cpp` | TA | RedNoteAdapter 全方法/滚动/去重/CDP 提取 |

## UT 测试点

### AdapterConfig
- 默认值全部正确
- 自定义值覆盖所有字段
- from_json 解析

### AdapterFactory
- `create()` 返回 RedNoteAdapter
- 未知平台 → 异常
- `llm_config` 透传

### extract
- 全字段映射
- 最小输入
- 备用键名回退

## TA 测试点

| 场景 | 验证 |
|------|------|
| CDP 全量提取 | N 条 UnifiedItem，字段正确 |
| limit 截断 | 仅返回 N 条 |
| 空页面 | 返回 0 条 |
| 去重 | 无重复 ID |
| CDP 滚动懒加载 | 有序输出 |
| health_check OK/fail | True/False |
| stealth 注入 | 正常运行 |
