# Adapter 测试方案

## 测试范围

| 文件 | 类型 | 用例数 | 覆盖内容 |
|------|------|--------|----------|
| `test_adapter_models.py` | UT | 6 | AdapterConfig 默认值/自定义/等值/URL 模板 |
| `test_adapter_factory.py` | UT | 4 | AdapterFactory create/未知平台/非config输入 |
| `test_rednote_impl.py` | TA | 24 | RedNoteAdapter 全方法/滚动/去重/边界/stealth/extract/parse_int |

## UT 测试点

### AdapterConfig
- 默认值全部正确（platform/URLs/scrolls/delay/timeout）
- 自定义值覆盖所有字段
- dataclass 等值比较
- `search_url` 中 `{keyword}` 可替换
- `user_url_template` 中 `{user_id}` 可替换

### AdapterFactory
- `create(None)` 返回 RedNoteAdapter（默认 platform）
- `create(AdapterConfig(platform="rednote", max_scrolls=5))` 配置透传
- `create(AdapterConfig(platform="douyin"))` → ValueError
- `create("not a config")` 使用默认配置（静默降级）

### extract
- 全字段映射（19 个字段，含 platform/fetched_at 自动填充）
- 最小输入（仅 item_id + content）
- 备用键名回退（`id` → `item_id`, `username` → `author_name`, `title` → `content_text`）

### _parse_int
- 中文万：`"1.2万"` → 12000
- 中文万非法：`"abc万"` → 0
- 英文 k：`"3.5k"` → 3500
- 英文 k 非法：`"xyzk"` → 0
- 逗号分隔：`"1,234"` → 1234
- 非法字符：`"not a number"` → 0
- 空字符串：`""` → 0
- 负数：`"-1"` → -1

### 测试页技术

- 使用 `file://` URL 加载本地 HTML 文件（`tmp_path`）
- 页面嵌入 JavaScript 模拟无限滚动：初始显示 N 条，每次 `scroll` 事件接近底部时追加 M 条
- 延迟设为 0.05s（加速测试），15 批上限
- 每项含完整 DOM 结构：cover img + title + author + content + like count + link
