## 2. 测试点清单

### UnifiedItem
- [x] 创建默认值 — 只给 `platform` + `item_id`，其余字段使用 dataclass 默认值
- [x] 创建全字段 — 所有字段显式赋值，包括 LLM 扩展字段
- [x] `to_dict()` — 完整字段输出为 dict，`content_media` 空列表不出错
- [x] `to_json()` — 输出 JSON 字符串，ensure_ascii=False
- [x] `from_dict(dict)` — 正常完整 dict 还原
- [x] `from_dict({})` — 空 dict，所有字段回退到默认值，`[]` 不为 None
- [x] `from_dict` 部分字段 — 只给 `platform` + `item_id`，其余取默认
- [x] `from_dict` null 值容错 — `content_media=None` → `[]`；`llm_tags=None` → `[]`；`metrics=None` → `{}`
- [x] 往返一致性 — `from_dict(to_dict())` 后字段一致

### TaskResult
- [x] 默认值 — 只给 `task_name`，数值字段均为 0/空
- [x] 全字段赋值 — fetched_count、kept_count、llm_calls、items 等

### RunStatus
- [x] 默认值 — running=False, current_task=None, progress=""
- [x] 运行态 — running=True, 含 current_task 和 progress
